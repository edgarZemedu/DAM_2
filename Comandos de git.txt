
-> ls -la    (para ver la lista de carpetas)
-> git log    (ver las últimas modificaciónes "--oneline")
-> git branch x    (crear una nueva rama o branch "master o x ")
-> git checkout x    (cambiar de rama o branch "master o x ")
-> git merge x    (para fusionar las dos ramas y desde la rama master)
-> git pull    (will pull the latest changes)
-> git fetch    (will update the list of changes)
-> git status    (will check the status of the repo. Without the fetch first, you will not see remote changes)
-> git config core.autocrlf = true
